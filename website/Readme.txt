This application logs your Guild Wars 2 character location by using the Mumble plugin API. Mumble is not required.

Source
https://github.com/codemasher/gw2api-tools gw2api-tools GitHub Repo
https://gw2apicpp.codeplex.com Location sender by Heimdall.4510 (source)
http://gw2.chillerlan.net/files/GW2LocationSender-setup.exe Application installer created by Smiley (binaries)

The location receiver (gw2location.php) was modified by me to fit my needs and the map (gw2map.html) was written mostly by me using the Leaflet library and the Guild Wars 2 map API.
